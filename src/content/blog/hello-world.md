## 欢迎来到我的博客

这是我使用 **Astro** 搭建的个人博客，部署在 GitHub Pages 上。

### 为什么选择 Astro？

- **零 JavaScript** - 默认不发送任何 JS 到浏览器
- **快速构建** - 毫秒级的构建速度
- **Markdown 支持** - 用 Markdown 写作，简单高效
- **SEO 友好** - 自动生成站点地图和 RSS

### 技术栈

这个博客使用了以下技术：

```
Astro + TailwindCSS + TypeScript + GitHub Pages
```
